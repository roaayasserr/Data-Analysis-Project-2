# Project-2-DA

## Description 
This project explores the relationship between listening to music while studying and its effect on focus and productivity. Data was collected using a survey and analyzed using Python.

## Dataset
The dataset was collected through a questionnaire and includes responses about:
- Study habits
- Music preferences while studying
- Self-reported focus levels


## Tools & Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Scikit-learn


## How to Run  
1. Open the notebook file (`notebook.ipynb`)
2. Run all cells step by step
3. Make sure required libraries are installed
